<h4> Spesifikasi Tugas : </h4>
<ul>
  <li> Membuat program dari algoritma Round Robin dengan menggunakan bahasa C </li>
  <li> Algoritma Round Robin digunakan untuk menjadwalkan proses secara adil </li>
  <li> Penjadwal Round Robin umumnya menggunakan pembagian waktu (time slot) untuk setiap pekerjaan atau kuantum </li>
  <li> Setiap proses yang dijadwalkan memiliki waktu kedatangan dan waktu eksekusinya masing-masing </li>
  <li> Sehingga input dari program ini adalah jumlah proses, waktu kedatangan dan waktu eksekusi untuk setiap proses, serta besar waktu kuantum </li>
  <li> Program akan berhenti jika telah diperoleh jadwal hasil dari algoritma Round Robin </li>
</ul>
